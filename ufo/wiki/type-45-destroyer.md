# Type 45 Destroyer

[[royal-navy|Royal Navy]] anti-air-warfare destroyer. Six in service as of 2024 (with fewer typically available for operations). The UK's only platform with any ballistic-missile detection / tracking / partial-intercept capability — track up to 200 miles, intercept up to 70 miles. Inherently maritime, so it cannot substitute for a [[ground-based-air-defence|GBAD]] capability.
