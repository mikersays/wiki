# Project Second Storey

1950s [[department-of-national-defence|DND]] committee tasked with reviewing UAP reports. Per the [[sky-canada-project]] final report ([[src-canada-sky-canada-final-2025-07]]), Second Storey ultimately recommended *ending* Canadian military UAP investigations — a precedent the Sky Canada Project final report cites when arguing that a future federal UAP function should be science-led and civilian rather than housed inside the defence ministry.
