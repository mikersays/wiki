# FBI Case File 62-HQ-83894

**62-HQ-83894** is a Federal Bureau of Investigation case file covering the years **1947–1968**, concerning unidentified objects and so-called "flying discs."

The file is the centerpiece of [[pursue]] **Release 01** (May 8, 2026). The PURSUE manifest CSV reveals **all 56 FBI records in Release 01 belong to this case** — 48 PDFs (sections, serials, and supplementary documents) plus 8 photographic-evidence images. Per the official description, this is the **complete case file** with newly declassified pages and only minor redactions, distinct from the partially-public FBI Vault version which has more redactions and missing pages.

## Notable Contents

- **High-profile incident accounts**, eyewitness testimonies, and public reports concerning UFOs and "flying discs" between June 1947 and July 1968.
- **Photographic evidence from sites like Oak Ridge, Tennessee** — a recurring DOE-site thread within the file.
- **Technical proposals regarding potential propulsion systems**.
- **Convention programs, researcher accounts, and extensive media coverage** from the period.
- **An FBI photo composite paired with the eyewitness sketch** of an "apparent ellipsoid bronze metallic object," 130–195 feet in length, that "materialized out of a bright light in the sky… and disappeared instantaneously" — corroborating eyewitness reports from a **September 2023** incident (Sketch dated 2024-04-30).
- The file is contemporary with the [[fbi]]'s broader Cold War-era UFO interest and overlaps the lifespans of Project Sign, Project Grudge, and Project Blue Book.
