# Ryugu (asteroid)

Carbonaceous near-Earth asteroid sampled by JAXA's Hayabusa2 mission (samples returned 2020). Subsequent organic-chemistry analyses have repeatedly identified prebiotic molecules — per [[src-loeb-panspermia-impact-survival-2026-03]], [[avi-loeb]] cites recent work identifying all five DNA bases in Ryugu material, which he treats as additional support for delivery-of-prebiotic-chemistry / [[panspermia]] arguments.
